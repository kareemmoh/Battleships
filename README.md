# Kickstart

Battleships is a small game written in java.
It will be equipped with the following features:

  - Games over LAN
  - .ini-file support

### Version

0.1.0

### Installation
Currently there is **no installer**. Simply pull the repository:

```sh
$ git clone [git-repo-url] battleships
```
